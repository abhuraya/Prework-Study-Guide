Git

  • Git is an online service to create and maintain repositories on the cloud
  • A user can make an account and then create repositories
  • Repositories can be downloaded and modified locally then uploaded
  • It allows multiple people to work on the same program at the same time
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • order to push to git hub
    1. git add -A
    2. git commit -m "First commit"
    3. git push
