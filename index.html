CSS

  • CSS augments HTML by adding customization
  • It is coded as a separate file and linked to the HTML file
  • Font color, size, and type are some of the available programmable
    features in CSS
  • Other features include background color and compartamentilization
  • A margin indicates how much space we want around the outside of an elemnet.
  • A padding indicates how much space we want around the content inside an element
  • A star indicates a universally applied feature
  • A period to start a feature indicates the feature will be applied to id's

Git

  • Git is an online service to create and maintain repositories on the cloud
  • A user can make an account and then create repositories
  • Repositories can be downloaded and modified locally then uploaded
  • It allows multiple people to work on the same program at the same time
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • order to push to git hub
    1. git add -A
    2. git commit -m "First commit"
    3. git push
